How to connect

1
Hold BOOT, plug USB
Some boards need the BOOT button held while connecting the USB cable to enter download mode. Check your board's pinout.
2
Click "Install PhantomRF"
A browser dialog will appear. Select the USB serial device that looks like USB JTAG/serial debug unit or CP210x / CH34x.
3
Wait for the flash to complete
Erase + write + verify takes 30–60 seconds. The device will reboot into PhantomRF and start broadcasting the PhantomRF WiFi network.
4
Connect and open the UI
Join the PhantomRF WiFi network (password printed on the serial monitor the first time). Browse to http://192.168.4.1.

Troubleshooting

"No serial port appeared"

  • Try a different USB cable — many are charge-only.
  • Check that the CH340 / CP2102 / USB-JTAG driver is installed for your board.
  • On Linux, add your user to the dialout group: sudo usermod -aG dialout $USER.
  • On macOS, allow the driver in System Settings → Privacy & Security.

"Failed to connect: wrong boot mode"

The ESP32-S3 is stuck in a non-download mode. Power-cycle, then hold BOOT while plugging the USB cable. Release BOOT once the device appears as a serial port.

"Flash succeeded but the device doesn't boot"

Hold BOOT, press and release RESET, then release BOOT. Re-run the flasher. If the issue persists, erase the flash first:

esptool.py --chip esp32s3 --port /dev/ttyACM0 erase_flash

"I want to flash over OTA later"

Once PhantomRF is running, use the firmware page inside the web UI (no cable needed). The device will reboot when the new firmware is verified.

Manual install

If the web flasher does not work, you can install the firmware using esptool.py or PlatformIO:

pip install esptool
esptool.py --chip esp32s3 --port /dev/ttyACM0 write_flash 0x0 PhantomRF-esp32s3-n16r8-v1.0.0.bin

Or with PlatformIO:

git clone https://github.com/.../PhantomRF
cd PhantomRF
pio run -e esp32s3-n16r8 -t upload

For ESP32-S3 native drag-and-drop UF2, hold BOOT, press RESET, release RESET, release BOOT. The board will appear as a USB drive. Drag firmware.uf2 onto it.
