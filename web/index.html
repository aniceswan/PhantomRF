Overview

PhantomRF is a research platform for the 2.4 GHz and sub-GHz ISM bands. The web UI is one of five independent control channels (USB CDC, WebSocket WiFi, BLE Serial, OLED + buttons, and station-mode WiFi). When a 2.4 GHz attack is active, the WebSocket will degrade — use the OLED menu or USB serial to stop the attack.

Keyboard shortcuts

  • g then d — go to Dashboard
  • g then s — go to Spectrum
  • g then t — go to Terminal
  • Esc — close any open modal
  • Tab — terminal command completion
  • ↑ / ↓ — terminal history

Attack categories

Bluetooth

AFH sweep + brute force on the 79 BT Classic channels; NOACK injection on BLE ad channels 37/38/39; CW jamming on BLE data channels.

WiFi

Deauthentication and disassociation frames on all 14 channels. Beacon flood with user-supplied SSID list. Probe request flood. PMKID capture (M1).

2.4 GHz

nRF24L01+PA+LNA constant carrier sweep. Configurable channel range, PA level, data rate, sweep method. Multi-radio parallel mode.

Sub-GHz

CC1101 OOK/FSK jamming. Spot / range / hopper / keyfob presets. Record and replay RAW bitstreams as Flipper-compatible .sub files.

Safety

PhantomRF is a research tool. Never use it on networks or devices you do not own or have explicit permission to test. Unauthorised jamming is illegal in most jurisdictions. The device is designed to be loud — keep PA levels conservative and sweep range narrow when learning.
